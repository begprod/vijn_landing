How do I start scanning?

Scanning with PT BBS is fast and easy. Just enter the address of your site—on the internet or a local network—and configure your scanning method: Развернуть ▼

  • Without confirmation of site ownership, you can scan via an agent. An agent is a small program that can be run without local installation—just open a secure executable file and you can start scanning. This is a convenient method when you cannot or do not want to make alterations to your site's root directory, or when your site is located behind a NAT (in a local network). Note that this method does not allow for anonymous scanning. All requests from the agent are tunneled from our service to the site being scanned via the computer of the user who initiated the scanning. This means that the computer (IP address) where the requests originated will be visible on the site side.
  • With confirmation of site ownership. For this method, a special HTML file must be placed in the root directory of the site that you want to scan. Our system will "see" the file and allow scanning to be initiated.

Что такое агент и для чего он нужен? Зачем могут понадобиться несколько агентов?

Агент — это небольшая open-source программа (безопасный исполняемый файл), не требующая установки на компьютере и особых привилегий. Агент нужен для сканирования сайтов в локальных сетях (за NAT) и в тех случаях, когда нет возможности подтвердить владение сайтом с помощью сертификата, например когда аудит проводят внешние эксперты, не имеющие доступа к серверам сайта. Агент создает туннель, через который передаются запросы от сканирующего сервера к проверяемому сайту. Развернуть ▼
